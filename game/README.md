Factory Game

Как запустить
1. Открой проект в NetBeans
2. Нажми F6 (Run)
3. Кликай на падающие ящики для сбора продукции

Функционал
- Анимация падающих объектов
- Сбор продукции по клику
- Учет производства и финансов
- Графический интерфейс
